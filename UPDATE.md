# 更新文档

## 2026-07-17 — 修复圈数不更新、比赛无法结算的问题

- **问题**:检查点判定要求在越线的单帧内同时满足符号翻转、在路面内、前进速度大于 0。过门瞬间打滑、漂移出路面或碰撞弹开会永久错过该检查点,导致圈数计数器卡死、结算画面永不出现。
- **修复**:`js/main.js` `updateRaceLogic()` 改为区间判定——玩家位于检查点之后 `[0, 间距)` 范围内且在路面附近向前行驶的任意一帧即判定通过,错帧后可自动补登;完赛后停止判定防止重复计圈。

## 2026-07-16 — 修复道路无法绘制的问题

- **问题**:道路网格三角形缠绕顺序错误,面法线朝下,被背面剔除导致路面不可见。
- **修复**:`js/world.js:108` 索引顺序由 `idx.push(a, b, c, b, d, c)` 改为 `idx.push(a, c, b, b, c, d)`,法线朝上,道路正常渲染和受光。
